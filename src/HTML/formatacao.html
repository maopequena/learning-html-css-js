Formatação

O HTML oferece vários elementos para definir o texto com um significado especial. Esses elementos são chamados de "elementos de formatação". Eis alguns:

  • Texto em negrito, <b>.
  • Texto importante, <strong>.
  • Texto em itálico, <i>.
  • Texto com ênfase, <em>. Um leitor de tela pronunciará as palavras dentro desse elemento com ênfase, dando mais importância para esse texto.
  • Texto menor, <small>.
  • Texto realçado, <mark>.
  • Texto excluído, <del>. "Strike-through".
  • Texto inserido, <ins>. Fica sublinhado.
  • H2O, texto subscrito. <sub>. Aparece como meio caractere, abaixo da linha normal, e geralmente com uma fonte menor. Este texto pode ser usado para fórmulas químicas.
  • WWW[1], texto superescrito. <sup>. Aparece como meio caractere, acima da linha normal, e geralmente com uma fonte menor. Este texto pode ser usado para notas de rodapé.

Comentários

Comentários HTML não aparecem no navegador, mas podem ajudar com o código-fonte.

Para inserir comentários no código, use a seguinte sintaxe: <!-- Comentário aqui! -->. O ponto de exclamação no início é obrigatório. É possível ocultar várias linhas ao mesmo tempo.

Use comentários para adicionar avisos ou lembretes, ocultar texto permanente ou temporariamente ou fazer depuração do código, já que se pode ocultar linha por linha até achar o problema.

<!-- AVISO: --> Este parágrafo contém um comentário!

No código, tudo o que está entre < e > aparece em cinza, e fica oculto da página da Web.

Códigos de computador e digitação do usuário

Elemento <kbd>

Esse elemento define uma digitação pelo teclado (keyboard input). O conteúdo nessa tag é exibido com uma fonte monoespaçada padrão no navegador.
P. ex. <kbd>Ctrl + S</kbd>.

Elemento <samp>

Esse elemento define o resultado de um programa executado no computador. O conteúdo nessa tag é exibido com uma fonte monoespaçada padrão no navegador.
P. ex. <samp>Arquivo não encontrado.
Pressione F1 para continuar.</samp>.

Elemento <code>

Esse elemento define um pedaço de código. O conteúdo nessa tag é exibido com uma fonte monoespaçada padrão no navegador.
P. ex. <code>x = 5; y = 6; z = x + y print(z)</code>.

Esse elemento não preserva espaços adicionais em branco ou quebras de linhas. Para corrigir isso, aninhe o elemento <code> em um elemento <pre>:

<pre>
                <code>x = 5;
                y = 6;
                z = x + y;
                </code>
</pre>

Elemento <var> para variáveis

Esse elemento define uma variável de progração ou de uma expressão matemática. O conteúdo nessa tag é exibido em itálico.
P. ex. The area of a triangle is:
1/2 x <var>b</var> x <var>h</var>,
where <var>b</var> is the base, and <var>h</var> is the vertical height.

Cores

Nomes de cores

Cores HTML são especificadas por nomes predefinidos ou por valores RGB, HEX, HSL, RGBA, ou HSLA. É possível definir as cores do texto (com o atribnuto color), do fundo (com background-color) ou da borda (com border-color).

Valores de cores

Para valores de cores é possível usar rgb, hex ou hsl. Ao usar rgba e hsla, adiciona-se um canal Alpha (ou transparência) com valores que vão de 0 a 1, que correspondem à opacidade de 0 a 100%.

rgb(255, 99, 71)

#ff6347

hsl(9, 100%, 64%)

rgba(255, 99, 71, 0.5)

hsla(9, 100%, 64%, 0.5)

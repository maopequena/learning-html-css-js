Planos de fundo no CSS

É possível adicionar planos de fundo a elementos HTML usando as propriedades a seguir.

Propriedades que estilizam planos de fundo

background-image, background-repeat (com repeat, no-repeat, repeat-x, repeat-y, round, space), background-position, background-attachment (com scroll, fixed, local), background-size, opacity, background-image, background-clip, background-origin e background-size.

Observação: ao usar a propriedade opacity para exibir transparência no fundo de um elemento, todos os elementos filhos herdam essa propriedade. Isso pode dificultar a leitura do texto.

Ao usar uma imagem no plano de fundo, ela não deve interferir com o texto.

Para economizar tempo e digitação, é possível juntar as propriedades background-color, background-image, background-repeat, background-attachment e background-position, na ordem, com "background". P.ex.:

background: #ffffff url("img_tree.png") no-repeat right top;

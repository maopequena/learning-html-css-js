Cabeçalhos

Cabeçalhos HTML são títulos ou subtítulos que se deseja exibir em uma página da Web.

Os códigos <h1> até <h6> definem os cabeçalhos, do mais importante para o menos importante.

Os navegadores adicionam automaticamente espaço em branco (uma margem) antes e depois de um cabeçalho.

Cabeçalhos são importantes porque os mecanismos de pesquisa os usam para indexar a estrutura e o conteúdo das páginas. Os usuários geralmente leem a página pelos cabeçalhos. Use cabeçalhos <h1> para o título principal, seguidos dos cabeçalhos <h2>, <h3> e assim por diante. USE APENAS PARA CABEÇALHOS, NÃO PARA DESTACAR UM TEXTO NA TELA.

Cada cabeçalho HTML tem um tamanho padrão, porém é possível definir o tamanho de qualquer cabeçalho com o atributo style, usando a propriedade CSS font-size. Por exemplo:

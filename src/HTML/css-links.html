Links no CSS

É possível estilizar os links de várias formas: com bordas, cor de fundo, cor do texto e muito mais.

Há ainda quatro estados que se pode configurar:

  • :link
  • :visited
  • :hover
  • :active

Ao definir estes estados, há uma ordem que DEVE SER RESPEITADA: :link > :visited > :hover > :active.

A propriedade text-decoration geralmente é usada para remover o sublinhado dos links (text-decoration: none).

Usando as propriedades background-color, padding e border/border-radius é possível fazer o link se parecer com um botão ou outros objetos.

A propriedade display também é importante.

Listas

Listas podem ser não ordenadas (bullet points) ou ordenadas (numeradas).

Exemplo de lista não ordenada

Uma lista não ordenada começa com a tag <ul> e cada item da lista começa com a tag <li>.

É possível definir o estilo do marcador dessa lista como: disc, circle, square ou none.

  • Café
  • Chá
  • Leite

Também é possível usar uma imagem como o marcador com a sintaxe style="list-style-image: url('sqpurple.gif');".

A propriedade list-style-position: inside/outside permite alterar o agrupamento do marcador.

É possível ainda estilizar as listas com cores usando CSS. Saiba mais.

Exemplo de lista ordenada

Uma lista ordenada começa com a tag <ol> e cada item da lista começa com a tag <li>.

É possível definir o estilo do marcador com o atributo type="N", em que N é um dos seguintes: 1 (padrão, números), A (letras maiúsculas), a (letras minúsculas), I (números romanos maiúsculos) ou i (números romanos minúsculos).

Por padrão, as listas ordenadas começam a contar do "1". Para alterar isso, use o atributo start="N", em que N é o número inicial.

  1. Café
  2. Chá
  3. Leite

Exemplo de lista de descrição

Também é possível criar listas de descrição, que são listas de termos com uma descrição de cada termo. A tag <dl> define esse tipo de lista, a tag <dt> define o termo (o nome) e a tag <dd> descreve cada termo.

Café
- bebida quente e escura
Leite
- bebida clara e fria

É possível aninhar uma lista em outra.

Um item de lista (<li>) pode conter uma nova lista ou outros elementos HTML, como imagens, links, etc.

Exemplo de lista não ordenada aninhada

  • Coffee
  • Tea
    • Black tea
    • Green tea
  • Milk

Exemplo de lista ordenada aninhada

  1. Coffee
  2. Tea
    1. Black tea
    2. Green tea
  3. Milk

Estilização de listas

É possível estilizar listas HTML não ordenadas de várias formas usando CSS. P.ex.: estilizar horizontalmente para criar um menu de navegação.

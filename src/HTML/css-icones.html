Ícones no CSS

A maneira mais fácil de adicionar ícones a uma página HTML é com uma biblioteca de ícones, como Font Awesome, Bootstrap ou Google Icons. É preciso copiar um link para adicionar à tag <head> da página. Depois, é só adicionar o nome da classe especificado a qualquer elemento inline, como <i> ou <span>.

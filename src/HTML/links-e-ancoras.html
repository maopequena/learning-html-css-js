Links e âncoras

Observação: os links desta página foram estilizados com CSS, através do arquivo Link-colors.css.


Links HTML, também chamados de hiperlinks, possibilitam que o usuário navegue entre as páginas do site. Ao clicar em um link, o usuário é enviado para outro documento HTML, ou outra página. Ao passar o mouse sobre um link, a seta se transforma em uma mãozinha. Um link não precisa ser um texto, pode ser uma imagem ou qualquer outro elemento HTML.

Sintaxe

<a href="url">texto do link</a>. href é o atributo mais importante da tag <a>, já que é ele que contém o destino do link. O "texto do link" é o que fica visível para o usuário final.

Cores padrão dos links

Isso pode ser alterado no CSS ou com o atributo style

  • Links não visitados aparecem sublinhados e em azul
  • Links visitados aparecem sublinhados e em roxo
  • Links ativos aparecem sublinhados e em vermelho

Atributo target

Por padrão, ao ser clicado, o link abre na janela ou aba atual. Este atributo altera esse comportamento com os seguintes valores:

  • _self: mantém o comportamento padrão, abrir na aba atual. Visite a W3Schools.com!
  • _blank: abre em uma nova janela ou aba. Visite a W3Schools.com!
  • _parent: abre no frame pai. Visite a W3Schools.com!
  • _top: abre no corpo cheio da janela. Visite a W3Schools.com!

Imagem como URL

Para usar imagens como links, coloque a tag <img> dentro da tag <a>.

É possível usar URLs absolutos ou relativos no atributo href. URLs absolutos são o endereço completo, URLs relativos não têm o http(s)://www e pertencem ao mesmo domínio da página atual.

Link para e-mail

Também é possível usar a sintaxe <a href="mailto:e-mail">texto do link</a> para abrir o programa de email do usuário, para que ele possa enviar um email para alguém.

Botão como link

Para usar um botão HTML como link, é preciso usar código JavaScript, que possibilita especificar o que acontece quando ocorre determinado evento, como ao clicar em um botão.

Atributo title

Este atributo define informações adicionais sobre um elemento. Geralmente o conteúdo desse atributo é exibido como uma dica ao passar o mouse sobre o elemento.

Âncoras para a mesma página com "id"

Usando o atributo id é possível criar âncoras para texto na mesma página. Ou use a sintaxe href="nome_da_página.html#id para adicionar um link para um id em outra página.

Link em iframe

Também é possível usar um nome especificado de um iframe para abrir o link em um quadro na página. Esse quadro deve ser definido logo abaixo do elemento <a> como <iframe src="link para o nome do arquivo a abrir" width=porcentagem name="nome definido no elemento>.
